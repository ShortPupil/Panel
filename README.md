## 识别算法

### 要求

只能按同一时针画图

### 基本运算

**边数**和**曲点**共同作为识别标准

曲点由**曲率的峰值**决定

![](/home/songzi/图片/截图/曲率.png)



| 曲率               | 几何特征 |
| ------------------ | -------- |
| 连续零曲率         | 直线段   |
| 连续非零曲率       | 弧线段   |
| 局部最大曲率绝对值 | 角点     |
| 局部最大曲率正值   | 凸角点   |
